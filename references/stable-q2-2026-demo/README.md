# Stable Q2 2026 Demo Reference

This folder is reserved for the previous stable BORK Internet Boutique Q2 2026 web-presentation as visual reference material.

## Status

The Q2 2026 presentation is a visual reference and stable demo only.

It may be used to study:

- brand DNA;
- premium visual atmosphere;
- color and material feeling;
- logo, pattern, and crest usage;
- mobile lessons;
- motion observations;
- executive presentation tone.

Brand assets from the old presentation may be added or reused only after separate confirmation.

## Not An Architecture Source

This reference is not an architecture source for the new platform.

Do not copy or treat as constraints:

- fixed 5-slide structure;
- old section order;
- old navigation;
- old loader;
- old layout system;
- old CSS classes;
- old JavaScript logic;
- plain HTML/CSS/JS as a required stack.

The new platform may use React, TypeScript, Vite, reusable sections, typed data contracts, new navigation, new motion, and new infographic patterns.

## Source Code Boundary

Old source code must not be imported into runtime code.

If archived source is ever added after explicit confirmation, it must live only in:

```text
references/stable-q2-2026-demo/archived-source/
```

Do not copy the old architecture into `src/`.

## Correct Use

Use this reference to preserve brand DNA while inventing a stronger, reusable, data-driven corporate web-presentation platform.
